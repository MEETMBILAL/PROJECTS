# Recipe Finder

A simple React app to search recipes by ingredients using the Spoonacular API.  
Features: diet filter, save favorites (localStorage), random recipe, "What can I cook?" button (zero missing ingredients).  

## Setup

1. Get an API key from [Spoonacular](https://spoonacular.com/food-api)
2. Copy `.env.example` to `.env` and add your key:  
   `VITE_SPOONACULAR_KEY=your_key_here`
3. Install dependencies: `npm install`
4. Run dev server: `npm run dev`
